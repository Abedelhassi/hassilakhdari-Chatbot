If you want to run this project on your own machine, follow these steps:

### 1. Clone the repository
```bash
git clone [https://github.com/Abedelhassi/hassilakhdari-Chatbot.git](https://github.com/Abedelhassi/hassilakhdari-Chatbot.git)

Bash

npm install

Bash

# Build the project

npm run start
